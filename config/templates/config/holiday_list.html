On this page are all the holiday days registered in the system. The system takes these holidays into account when calculating the end-dates for lendings. Adding a holiday also triggers a recalculation of all existing lendings that weren't late on the start-date of the holiday.

Fine Skip Days

By default, holiday days count regularly for fines (as do weekend days).
This behavior can be changed by setting Skipped for Fine to true on a holiday. The system then skips the days in that holiday when counting the number of late-days.
