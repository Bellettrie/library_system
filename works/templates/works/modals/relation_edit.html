How to use this form

You should read the relation in this panel as 'from_work' 'relation' 'to_work'.
Example: "Two Towers" "is Part of Series" "Lord of theRings".
If the 'from' and 'to' are in the wrong order, then press the double arrows to change the direction.
When editing an existing relation, the 'from_work', 'to_work', 'relation_kind' are all locked. If you want to change those, instead remove the relation and add a new one.

Relation Kinds

The system currently supports three relation kinds:

SubWork

indicates that one work is physically a part of the other. This is used for instance to track omnibusses. Example: "Ultimate Hitchhikers Guide to the Galaxy".

Part of Series

This indicates that one work is a part of a series. Example: "The Wheel of Time".

Part of Secondary Series

A secondary series is a series that is not used for generating book-codes or other orderings. It may be used for things such as alternative reading orders. {% endcomponent %} {% endblock %}
